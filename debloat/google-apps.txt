# debloat/google-apps.txt — Google preinstalled apps (opt-in category)
#
# This list is loaded by default when you run `./debloat.sh`. If you use any
# of these apps, pass `--category lge,social,games` (whichever you DO want)
# and leave google-apps out — or list them in `--skip-categories google-apps`.
#
# IMPORTANT: removing these saves RAM but breaks features the rest of Android
# may expect (e.g., Assistant, Voice, Gmail sync). Only remove what you
# actively don't use. All removals are reversible with:
#   adb shell cmd package install-existing <package>
